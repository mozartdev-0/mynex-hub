# 🌐 Mynex Hub
> The future of Open Source development, powered by Next.js and Supabase.

## 🚀 Sobre a Mynex
A **Mynex** é uma iniciativa focada em criar ferramentas robustas e seguras para desenvolvedores. Nosso objetivo é transformar a maneira como projetos open source são construídos e compartilhados.

## 🛠️ Tecnologias
* **Framework:** Next.js 15 (App Router)
* **Estilização:** Tailwind CSS
* **Banco de Dados:** Supabase
* **Hospedagem:** Square Cloud (Plano Hobby)
* **OS de Desenvolvimento:** Zorin OS

## 📂 Estrutura do Projeto
Atualmente focado no diretório `app/` para máxima performance e roteamento moderno.

## ⌨️ Como rodar o projeto
1. Instale as dependências: `npm install`
2. Rode o servidor de desenvolvimento: `npm run dev`
3. Acesse: `http://localhost:3000`
